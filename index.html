Basic CSS

The full meaning of CSS is Cascading Style Sheet. It tells browser how to display the content you write in the html document.

CSS allows you to control:

  • color
  • fonts
  • positioning
  • spacing
  • sizing
  • decoration
  • transition

Types of CSS Styling

You can use CSS styling in three main ways. Which includes inline, internal and external. Inline CSS applied using style attribute. Internal CSS is used by using style tag in the head element and using CSS selectors and properties and values. And External CSS is used by creating external style sheet.
It is a good practice to use external CSS. You can target html element in the html DOM(Document Object Model) by using css selectors such as class, id, tag, pseudo classes. External CSS will need to be linked using link tag and using src="" attribute.
